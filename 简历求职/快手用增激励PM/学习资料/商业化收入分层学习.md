# 商业化收入分层学习：海外用户价值分层

## 1. 海外用户总价值体系与公式

基于国内研究，采用「现金率」概念，以广告收入定义商业化价值，并结合品牌广告、第三方广告形成**海外商业化价值**；再叠加直播、电商收入，构成**海外用户总价值体系**，即三大板块现金（或利润）水平的加总。

**总价值公式：**

Total Value =
（内循环效广消耗_互娱 × 现金率_互娱 0.65）+
（内循环效广消耗_非互娱 × 现金率_非互娱 0.72）+ 
品牌广告消耗_折后 + 
外部（第三方）广告消耗 +
（电商 GMV × tr_电商 0.2）+
（直播流水 × tr_直播 0.33）

### 关键参数说明


| 项目       | 说明 / 取值         |
| -------- | --------------- |
| 现金率（游戏）  | 65%             |
| 现金率（非游戏） | 72%             |
| 品牌广告     | 设备维度支出按广告曝光占比拆分 |
| 第三方广告    | 设备维度支出按广告曝光占比拆分 |
| tr_电商    | 20%             |
| tr_直播    | 33%             |

## 2. 用户分层方法

**分层逻辑（两步）：**

1. **价值刻画**：先按上文公式，在统计窗口内算出**每个设备**的**总现金价值**（单一标量）。
2. **分位分层**：再对该指标做**分位排序**与**等频分层**——从低到高排序后，切成五档，每档约 **20%** 用户。

在此基础上：

- 以**近 30 日活跃用户**为对象，按**人均 / 设备维度的平台现金价值**从低到高排序，均分为五档，每档约 **20%** 用户。
- **高价值用户（W5）**：现金价值最高的 **20%**。
- **最高价值用户（W5a）**：即全量用户中的前5%。

### 价值贡献（典型结论）


| 分层        | 对平台总价值贡献  | 分业务线结构（示例）          |
| --------- | --------- | ------------------- |
| W5a（前 5%） | 约 **81%** | 商业化 96%，电商 2%，直播 2% |
| W5（前 20%） | 约 **94%** | 商业化 96%，电商 2%，直播 2% |
