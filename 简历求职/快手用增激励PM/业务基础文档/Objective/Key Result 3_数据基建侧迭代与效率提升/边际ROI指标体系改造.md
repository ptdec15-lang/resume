# 边际ROI指标体系改造 PRD文档

---

## 一、需求背景与目标 

### 1.1 业务背景
* **现状问题**：
  - 现有数据看板基于原有指标体系搭建，涵盖 delta ROI、delta 成本、delta 收入、delta DAU、Delta DAU 金币成本/$ 等指标
  - 当前 Delta DAU 金币成本/$ 含维系、商业化、小游戏金币成本，无法单独观测激励业务（维系）的成本效率
  - 维度单一，缺少用户分层、设备分层等精细化分析能力
  - 边际收益计算需依赖 holdout 实验，需明确数据口径与分组逻辑

### 1.2 核心目标
* **业务指标**：建立以激励边际收益为核心的指标体系，支撑精细化效率决策
* **产品目标**：将原有数据看板改造为以边际 ROI 为核心的指标体系，支持多维度筛选与收益归因分析

---

## 二、术语说明 

* **边际ROI**：每增加一单位投入（如成本、资源）所带来的增量收益回报率，用于衡量投入的边际效益
* **Delta 指标（边际指标）**：基于对比产生的增量指标，数据本质为可归因的因果数据，依赖实验组 vs 对照组的对比结构
* **ROI**：投资回报率（Return on Investment），收益与成本的比值
* **Holdout 实验**：通过分组设计观测业务收益的对照实验，用于计算边际效应
* **维系成本**：激励业务投入的金币成本，不含商业化广告成本、小游戏金币成本

## 三、Holdout 实验与数据基础 

### 3.1 实验分组设计

| 序号 | 分组名称 | 分组描述 |
| :--- | :--- | :--- |
| 1 | base_blank（对照组） | 全局空白，没有 push、RTB、edm、激励 |
| 2 | exp1（自然组） | 屏蔽 push、RTB、EDM，仅保留激励，观测激励收益 |
| 3 | exp2（自然组） | 无屏蔽，观测全局增长收益 |

### 3.2 边际收益计算口径

**激励业务以 exp1 − base_blank 作为边际收益的计算基础**：

| 对比 | 含义 | 用途 |
| :--- | :--- | :--- |
| exp1 − base_blank | 激励单独带来的增量收益（排除 push、RTB、EDM） | 激励边际 ROI、激励边际收益 |
| exp2 − base_blank | 全量运营的边际收益 | 全局收益评估 |
| exp2 − exp1 | 非激励手段（push、RTB、EDM）的边际收益 | 非激励归因 |

## 四、现状分析 

### 4.1 原有指标体系概览

| 指标名称 | 计算口径 | 当前用途 | 存在问题 |
| :--- | :--- | :--- | :--- |
| Delta ROI | exp1-base 收益/成本 | 激励效率评估 | 已覆盖 |
| Delta 成本 | exp1-base 成本差值 | 成本监控 | 已覆盖 |
| Delta 收入 | exp1-base 收入差值 | 收益监控 | 已覆盖 |
| Delta DAU | exp1-base DAU 差值 | 活跃增量 | 已覆盖 |
| Delta DAU 金币成本/$ | 含维系+商业化+小游戏 | 成本效率 | 含非激励成本，无法反映纯维系效率 |

### 4.2 改造诉求
- 剔除商业化、小游戏金币成本，单独观测维系（激励）成本效率
- 补充收益类 delta 指标（ARPU、成本收入比、激励效率等）
- 增加多维度筛选能力，支撑精细化运营

## 五、改造方案 

### 5.1 收益类 Delta 指标补充

基于《维系指标体系业务洞察》，以下指标可作为 delta 指标供观测：

| 指标名称 | 业务意义 | 计算口径 | 优先级 |
| :--- | :--- | :--- | :--- |
| delta_ARPU | 单用户收入变化，与 AEPU 平衡 | (exp1 收入 − base 收入) / 对应 DAU | P0 |
| delta_激励效率 | 单位维系成本带来的 DAU 增量 | delta_DAU / delta_维系成本 | P0 |
| delta_留存率 | 激励对留存的影响 | 各周期（次留/3留/7留/14留/30留）的 delta | P1 |

## 六、维度设计 

### 6.1 新增维度清单

| 优先级 | 维度 | 业务价值 |
| :--- | :--- | :--- |
| P0 | 高低价值用户 | 支撑「精准激励」策略，识别高/低 ROI 用户 |
| P0 | 活跃分层 | 区分高/中/低活，优化激励策略 |
| P0 | 设备类型 | 分析不同设备表现 |
| P0 | 细分实验 | 拆分不同实验的收益归因 |
| P1 | 设备来源 | 新设备/回流设备/大盘设备，对应 DAU 拆解逻辑 |
| P1 | 生命周期分层 | 新用户/成长期/成熟期/流失期 |
| P2 | 功能渗透分层 | 签到/视频/任务用户，验证各功能激励效果 |
| P2 | 留存周期 | 次留/3留/7留/14留/30留 |
| P2 | 渠道 | 区分不同获客渠道的激励效率 |
