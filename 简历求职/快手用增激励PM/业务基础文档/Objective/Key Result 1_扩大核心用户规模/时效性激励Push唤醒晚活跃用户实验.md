# 时效性激励Push唤醒晚活跃用户实验

## 【S - Situation 业务背景】

**痛点诊断**：
1. **晚活跃/不活跃用户存在唤醒机会**：部分用户在每日活跃周期中呈现「晚活跃」或「不活跃」特征，距离每日结束仍未活跃的用户群体，存在显著的当日唤醒空间
2. **过往验证**：基于用户行为分析和过往实验数据，精准触达 + 时效性激励相结合的方式，可有效提升该群体的当日活跃率

**业务本质**：
- 晚活跃用户 = 当日尚未活跃但仍有唤醒可能 → 精准触达可提升当日DAU
- 时效性激励 = 制造紧迫感 → 提升点击与转化
- 触达时机 = 当日临近结束前的关键窗口，需验证最优时间点（T-4 vs T-6）

---

## 【T - Task 实验目标】

- **当日活跃提升**：唤醒当日尚未活跃的用户，提升大盘DAU
- **高价值用户聚焦**：重点提升高价值设备（R4+）的活跃
- **成本与体验可控**：AEPU、Push关闭率符合护栏指标约束
- **触达时机选型**：通过T-4 vs T-6双组对比，观察时间占比与敏感性，找到最优触达窗口

---

## 【A - Action 策略与执行】

**核心策略**：针对当日临近结束仍未活跃的用户，推送时效性激励消息，采用稀缺性文案 + 大额落地页 + 自动消失机制，提升触达转化与用户体验

**目标人群**：当日截止时间点尚未活跃的用户

**实验设计（双组对比）**：

| 实验组 | 触达时机 | 说明 |
|:---:|:---|:---|
| **Exp1** | T-4小时（距当日结束前4小时） | 较晚触达，用户剩余活跃时间短 |
| **Exp2** | T-6小时（距当日结束前6小时） | 较早触达，用户剩余活跃时间更长 |

**观测维度**：
- **时间占比**：不同触达时机下，目标人群在当日活跃用户中的占比，以及触达后的转化占比
- **敏感性**：T-4 vs T-6 两组在DAU、点击率、转化率等指标上的差异，验证用户对触达时机的敏感程度

**策略设计**：

| 要素 | 设计 | 业务逻辑 |
|:---|:---|:---|
| **触达时机** | T-4 / T-6 双组验证 | 找到最优触达窗口，平衡覆盖与转化 |
| **激励形式** | 限时金币奖励（1小时内领取有效） | 时效性制造紧迫感 |
| **落地页** | 高转化率的大额Push落地页 | 提升点击→转化的承接效率 |
| **消息机制** | 1小时后从通知栏自动清除 | 避免长期堆积，降低打扰感 |
| **文案策略** | 稀缺性 + 损失厌恶 | 行为经济学原理，吸引用户点击 |

**心理学原理**：
- **稀缺性**：限时有效 → 制造紧迫感
- **损失厌恶**：不领就没了 → 提升点击意愿

---

## 【R - Result 实验结果】

**核心结论**：高价值DAU显著提升，AEPU和Push关闭率符合约束指标，**申请推全**。

### 关键数据表现

| 指标 | 变化 | 说明 |
|:---|:---:|:---|
| 大盘DAU | **+0.047%** | 当日活跃正向增长 |
| 高价值R4+设备数 | **+0.274%** | 高价值用户唤醒效果显著 |
| AEPU | +0.086% | 成本可控，符合约束 |
| 通知关闭率 | **-0.188%** | 用户反感度下降，体验改善 |
| 人均展现消息数 | +0.157% | 触达量提升 |

**T-4 vs T-6 对比观察**：

| 观测维度 | 说明 | 待补充 |
|:---|:---|:---|
| **时间占比** | 不同触达时机下，目标人群占比及触达后转化占比 | T-4 / T-6 各组数据 |
| **敏感性** | 两组在DAU、点击率、转化率等指标上的差异 | 用户对触达时机的敏感程度 |

**业务价值**：
- 验证「晚活跃用户 + 时效性激励」的唤醒有效性
- 高价值用户（R4+）提升幅度大于大盘，精准触达策略有效
- 成本与体验双指标均在护栏内，可持续推全
- 双组对比为后续触达时机优化提供选型依据

---

## 后续迭代

**方向**：支持更加灵活的配置项

- **增发条数**：可配置单日增发Push条数
- **增发间隔**：可配置多条消息的发送间隔
- **目标**：通过灵活配置获取更多DAU收益，同时控制触达频次避免过度打扰
